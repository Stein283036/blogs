# 集合框架

## 概述

集合框架指的是 java.util 包下的以 Collection 和 Map 两大接口展开的一系列集合相关类的实现，集合类注释上都会标注这个类是集合框架的一部分。

集合顾名思义用来存储多个元素的值，且只能存储引用数据类型的元素，即只能存储对象，区别与数组既可以存储基本数据类型元素也可以存储引用类型元素，但是数组在只是 Java 语言层面的支持，使用起来没有集合类方便，添加元素时需要亲自移动数组元素的位置、数组容量不足时需要亲自手动扩容并拷贝原数组等等一系列的操作不便，而集合基于面向对象的设计，提供了很多 API 对集合进行修改，如 添加元素、删除元素、排序元素、查找元素、比较元素、获取集合大小、动态扩容等等，且 JDK 的集合框架实现效率很高。

Java 中的集合分为两大类，即 Collection 单列集合 和 Map 双列集合；而 Collection 又有两大子接口 List 和 Set。其中 List 是顺序允许重复元素的集合，Set 是不允许重复默认无序的集合。List 的主要实现类是 ArrayList 和 LinkedList，Set 的主要实现类是 HashSet，Map 的主要实现类是 HashMap。
